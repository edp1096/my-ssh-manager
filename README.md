SSH host manager for personal using

## Usage
```sh
./ssh-manager
# or
./ssh-manager &
```

## Requirements
* MS-Windows
    * Windows Terminal
    * chrome or ms-edge
* FreeBSD, Linux
    * gnome-terminal or konsole or xterm
    * tmux
    * chromium or chrome

## First run
* Password
    * First time - create `hosts.dat` with entered password
    * Then - Password is required when `hosts.dat` open

## Build
```sh
make
```
